Pourquoi c'est pédagogiquement puissant ? Cette architecture est exactement celle utilisée dans les systèmes d'IA embarqués professionnels (véhicules autonomes, robotique, caméras intelligentes). La Hailo-8 détecte les objets à 45 FPS, le LLM explique le code algorithmique — tout ça sur un Pi de la taille d'une carte de crédit !
Objectif du projet Assistant Web IA complet · 100 % hors-ligne
  1. Génère du code Python NSI via Ollama (CPU ARM)
  2. Détecte des objets en temps réel via Camera Module 3 + Hailo-8 (YOLOv8)
  3. Diffuse un flux vidéo annoté vers les navigateurs des élèves
  4. Fonctionne 100 % hors-ligne — sans Internet, sans box
Niveau requis Terminale NSI · 5 à 7 séances de 2h
Terminale NSI Python HTML de base Linux CLI

Notions requises : Python, HTML de base, lignes de commande Linux.

Durée estimée : 5 à 7 séances de 2 heures.
