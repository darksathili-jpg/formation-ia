NSI : Générateur de Code Python

Pour utiliser cette application, vous devez disposer d'une clé API Groq : https://console.groq.com/home. Veuillez entrer votre clé pour commencer à générer du code Python. La clé est sauvegardée dans localStorage (thread principal) et utilisée pour les requêtes vers le worker IA. Elle n'est pas envoyée à un serveur externe. Au démarrage de l'application, la clé est restaurée depuis localStorage.
