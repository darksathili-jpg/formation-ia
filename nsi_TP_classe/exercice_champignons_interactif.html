Requêtes SQL — Base de données Champignons

Une base de données relationnelle stocke des données dans des tables (attributs = colonnes, tuples = lignes).

Clé primaire : identifie de façon unique chaque ligne — soulignée dans le schéma.
Clé étrangère (#) : référence la clé primaire d'une autre table.

SELECT — lire et filtrer :

SELECT nom FROM champignon WHERE lamelle = 'oui' AND couleur = 'orange';

JOIN — jointure entre deux tables :

SELECT c.nom
FROM   champignon c
JOIN   ordre o ON c.id_ordre = o.id
WHERE  o.classe = 'agaricomycetes';

JOIN double — jointure entre trois tables :

SELECT c.nom
FROM   champignon c
JOIN   ordre o    ON c.id_ordre    = o.id
JOIN   toxicite t ON c.id_toxicite = t.id_tox
WHERE  o.nom = 'amanitales' AND t.type = 'tres toxique';

INSERT INTO — ajouter une ligne :

INSERT INTO champignon(id, nom, id_ordre, lamelle, couleur, chapeau_min, chapeau_max, pied_min, pied_max)
VALUES (56, 'amanite solitaire', 4, 'oui', 'blanc', 6, 20, 4, 10);

UPDATE — modifier une valeur :

UPDATE champignon
SET    id_toxicite = 1
WHERE  nom = 'amanite citrine'; -- sans WHERE : TOUTES les lignes seraient modifiees !

Partie B

Programmation objet — Classe Champignon

La POO regroupe données (attributs) et comportements (méthodes) dans des objets créés à partir d'une classe.

class Animal():
    def __init__(self, nom, age):
        self.nom = nom      # attribut d'instance
        self.age = age

chat = Animal('Felix', 3)
print(chat.nom)   # -> Felix
Points de vigilance pour cet exercice :
  • Le nom du paramètre peut différer du nom de l'attribut : __init__(self, lieu) stocke self.localisation = lieu. On accède à obj.localisation, jamais obj.lieu.
  • return est réservé aux fonctions. Dans une boucle hors d'une fonction, Python lève une SyntaxError.
  • Comparaison de chaînes : c.cuisson vaut '12 minutes a feu moyen', pas 'feu moyen' seul. L'opérateur == exige l'égalité exacte.
Fonction recherche_textuelle(texte, mot) fournie par Iris :
Renvoie True si mot est un mot entier contenu dans texte.

recherche_textuelle('dans une foret', 'foret') → True
recherche_textuelle('pres de la mer', 'mare') → False
Classe Champignon fournie dans le sujet (Partie B) 
class Champignon():
    def __init__(self, nom, lieu, saison, recette, cuisson):
        self.nom          = nom
        self.localisation = lieu      # parametre 'lieu' -> attribut 'localisation'
        self.saison       = saison
        self.recette      = recette
        self.cuisson      = cuisson
Instanciation exemple donnée dans le sujet 
champignon1 = Champignon('Lactaire delicieux', 'Sous les pins',
    'ete', "Lactaires grilles a l'huile d'olive", '12 minutes a feu moyen')
