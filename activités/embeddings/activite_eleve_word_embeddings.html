Partie 4 — Visualisation 2D

Projeter les vecteurs dans un plan PCA — Analyse en composantes principales

Les vecteurs ont 50 dimensions — impossible à visualiser directement ! On utilise une technique mathématique (PCA) pour les projeter en 2D tout en conservant l'essentiel de l'information.

import matplotlib.pyplot as plt from sklearn.decomposition import PCA mots = ["cat", "dog", "lion", "tiger", "car", "truck", "bus", "paris", "queen", "man", "woman"] vecteurs = [modele[m] for m in mots] # Réduction en 2 dimensions pca = PCA(n_components=2) coords = pca.fit_transform(vecteurs) # Affichage plt.figure(figsize=(9, 6)) for i, mot in enumerate(mots): x, y = coords[i] plt.scatter(x, y, s=80) plt.annotate(mot, (x, y), fontsize=11, xytext=(4, 4), textcoords='offsetpoints') plt.title("Word Embeddings — projection 2D (PCA)") plt.grid(True, alpha=0.3) plt.tight_layout() plt.show()
