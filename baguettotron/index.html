Baguettotron

Baguettotron est un modèle d'intelligence artificielle développé par PleIAs, startup française basée à Paris. Il incarne une approche éthique et souveraine de l'IA : entraîné exclusivement sur des données ouvertes et libres de droits, respectueux du droit d'auteur, et économe en énergie.
PleIAs a été remarquée par WIRED, Euronews et Les Echos pour avoir prouvé qu'il est possible d'entraîner un modèle IA performant sans utiliser de données protégées par le droit d'auteur.

Évalué sur trois benchmarks industriels majeurs :
MMLU — Raisonnement général GSM8K — Mathématiques HotPotQA — Retrieval
Avec seulement 321M de paramètres, Baguettotron approche les performances de Qwen-0.6B et surpasse nettement Gemma de taille comparable — tout en ayant été entraîné sur considérablement moins de données.
Il offre un équilibre inédit entre mémorisation, raisonnement général, mathématiques et retrieval pour un SLM de sa catégorie.

Le modèle a été entraîné sur des pipelines dédiés couvrant les tâches suivantes :
  • Mémorisation encyclopédique (50 000 articles Wikipedia)
  • Génération augmentée par retrieval (RAG)
  • Résolution de problèmes arithmétiques et mathématiques simples
  • Tâches d'édition et réécriture
  • Extraction d'information
  • Écriture créative

Baguettotron peut lire et écrire dans les principales langues européennes :
Français Allemand Italien Espagnol Polonais Latin (partiel)
Les traces de raisonnement (chain-of-thought) sont exclusivement générées en anglais.

Comme tout modèle compact, Baguettotron présente certaines limites à prendre en compte :
Non adapté au code Mémorisation réduite sans chain-of-thought Mathématiques complexes limitées
Pour des tâches de génération de code ou de raisonnement très complexe, des modèles plus larges restent recommandés.

Cette version utilise l'API Inference Serverless de Hugging Face. Aucun serveur local requis.
  • Créez un compte sur huggingface.co
  • Générez un token dans Settings → Access Tokens
  • Collez le token dans le champ prévu dans les paramètres
L'API Inference HF est gratuite avec quota journalier. Pour plus de requêtes, un abonnement PRO HF (~9$/mois) est disponible.
