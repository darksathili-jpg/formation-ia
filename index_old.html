Prompt Engineering : DarkSATHI Li

Formation en prompt engineering
Le prompt engineering est la pratique qui consiste à concevoir, tester et améliorer les consignes que l’on donne à une IA (texte, images, code, etc.) pour la guider vers le résultat souhaité, de la façon la plus fiable et efficace possible.
Concrètement, il s’agit de :
  • Décrire clairement la tâche (contexte, objectif, format de sortie).
  • Donner des contraintes (ton, langue, longueur, public visé).
  • Fournir éventuellement des exemples de bonnes réponses à imiter.
  • Ajuster le prompt par essais/erreurs pour corriger les réponses de l’IA.

La méthode RCTF

Structurer un prompt pédagogique efficace // Les 4 ingrédients d'un prompt réussi
R
Rôle
"Tu es un prof de NSI Terminale…"
C
Contexte
"…pour 25 élèves de niveau intermédiaire…"
T
Tâche
"…génère un exercice sur l'algorithme BFS…"
F
Format
"…avec graphe ASCII et corrigé Python commenté."
# Exemple de prompt complet "Tu es un professeur de NSI Terminale. Pour une classe de 25 élèves niveau intermédiaire, génère un exercice sur Dijkstra : graphe ASCII 6 nœuds, 3 questions guidées, corrigé Python commenté avec complexité."
Astuce : Si le résultat ne convient pas, itérez — précisez davantage. L'IA améliore sa réponse à chaque reformulation du prompt.
